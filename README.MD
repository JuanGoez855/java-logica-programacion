## Instrucciones de Ejecución

### Requisitos previos
* Java JDK 11 o superior instalado.
* Un IDE (IntelliJ IDEA, Eclipse, VS Code) o acceso a la terminal.

### Compilación y Ejecución (Consola)
1. Navega hasta la ruta raíz del proyecto: `src/main/java`.
2. Compila el archivo principal:
   ```bash
   javac com/example/App.java
